Що робить імпорт
  • Знаходить студента нечітким пошуком по ПІБ

Формат файлу

Перший рядок — заголовки (ігноруються). Дані починаються з рядка 2.
Файл містить рівно дві колонки.

# Колонка Обов'язкова Приклад
A ПІБ студента Так Шевченко Іван Олександрович
B Рядок документа Так Диплом бакалавра B24 088726; 20.06.2024; Ким видано: ...
C Академічна довідка (укр) Ні 08-17/04; ТОВ Кавказький університет; Грузія; 02.06.2017
D Академічна довідка (en) Ні 08-17/04; Caucasus University LLC; Georgia; 02.06.2017
E Свідоцтво про визнання (укр) Ні № N-7916-20; Приватний ВНЗ «Європейський університет»; 12.05.2020
F Свідоцтво про визнання (en) Ні № N-7916-20; Private HEI "European University"; 12.05.2020
{# Формат рядка документа #}
Формат рядка документа (колонка B)
[Тип]  [Номер]  ;  [Дата ДД.ММ.РРРР]  ;  Ким видано: [Заклад]

Між типом документа і номером — два або більше пробіли. Між частинами — крапка з комою (;).

Формат колонок C та D (Академічна довідка)
C: [Номер довідки] ; [Заклад укр] ; [Країна укр] ; [Дата видачі]
D: [Номер довідки] ; [Institution EN] ; [Country EN] ; [Дата видачі]

У колонці D номер та дата дублюються — використовуються лише заклад та країна англійською.

Формат колонок E та F (Свідоцтво про визнання)
E: [Номер свідоцтва] ; [Орган укр] ; [Дата видачі]
F: [Номер свідоцтва] ; [Issuer EN] ; [Дата видачі]

У колонці F номер та дата дублюються — використовується лише орган англійською. Колонки C–F необов'язкові.
