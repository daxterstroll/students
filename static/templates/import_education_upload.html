Що робить імпорт
  • Знаходить студента нечітким пошуком по ПІБ
  • Розбирає рядок документа на поля автоматично
  • Розбирає (якщо вказані) академічну довідку та свідоцтво про визнання
  • Змінює . на / в усіх датах
  • Визначає країну та автоматично перекладає заклад/орган англійською

Формат файлу

Перший рядок — заголовки (ігноруються). Дані починаються з рядка 2.
Файл містить від двох до чотирьох колонок. Переклад англійською (тип документа, заклад, довідка, свідоцтво) виконується автоматично — окремих колонок для англійських назв у файлі не потрібно.

# Колонка Обов'язкова Приклад
A ПІБ студента Так Шевченко Іван Олександрович
B Рядок документа Так Диплом бакалавра B24 088726; 20.06.2024; Ким видано: ...
C Академічна довідка Ні 08-17/04; ТОВ Кавказький університет; Грузія; 02.06.2017
D Свідоцтво про визнання Ні № N-7916-20; Приватний ВНЗ «Європейський університет»; 12.05.2020
{# Формат рядка документа #}
Формат рядка документа (колонка B)
[Тип]  [Номер]  ;  [Дата ДД.ММ.РРРР]  ;  Ким видано: [Заклад]

Між типом документа і номером — два або більше пробіли. Між частинами — крапка з комою (;).

Формат колонки C (Академічна довідка)
[Номер довідки] ; [Заклад укр] ; [Країна укр] ; [Дата видачі]

Заклад та країна англійською перекладаються автоматично.

Формат колонки D (Свідоцтво про визнання)
[Номер свідоцтва] ; [Орган укр] ; [Дата видачі]

Орган англійською перекладається автоматично. Колонки C і D необов'язкові.

Частина Опис Приклад
Тип документа Назва документа Диплом бакалавра
Номер Серія та номер (через пробіл) B24 088726
Дата Дата видачі ДД.ММ.РРРР 20.06.2024
Ким видано Назва закладу (визначає країну) Київський університет ім. Шевченка

Приклади рядків
  • Диплом бакалавра B24 088726; 20.06.2024; Ким видано: Харківський національний університет
  • Атестат про повну загальну середню освіту БК 40920464; 15.06.2021; Ким видано: Ліцей №5 м. Одеса
  • Диплом молодшого спеціаліста КМ 12345678; 30.06.2022; Ким видано: Київський коледж технологій
{# Причини пропуску #}
Коли рядок позначається як помилковий
  • Студента з таким ПІБ не знайдено (або схожість менше 80%)
  • Рядок документа не відповідає формату (відсутній ; або Ким видано:)
  • ПІБ або рядок документа (колонки A/B) порожні
